# 备份恢复必须掌握知识点（对应 questions-engineering 题 4）

## 1. 备份的目标

### 概念

- 备份不是“做了就行”，必须可恢复、可验证、可演练。

## 2. 常见备份策略

### 全量 + 增量/日志

- 全量备份：每天/每周。
- 增量备份：基于 binlog 或增量工具，缩短 RPO。

### 保留周期

- 在线保留 + 离线归档。
- 需要考虑合规与成本。

## 3. 恢复与演练

### 必会要点

- 定期做恢复演练：验证备份链路与操作手册。
- 校验备份可用性：校验文件完整性、抽样恢复。

### 常见坑

- 只备份不演练，真正故障时恢复步骤缺失/时间不可控。

## 对应题目

- 题 4

## 考核要点/验收清单

- **验收项**：能描述一套备份策略（全量/增量或 binlog、保留周期、存储介质）。
- **验收项**：能明确 RPO/RTO 的目标（或至少说明如何影响备份频率与演练频率）。
- **验收项**：能给出恢复演练的步骤要点，并说明“只备份不演练”的风险。
