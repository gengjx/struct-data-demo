# 数据结构与算法练习

本模块包含了数据结构和算法的完整练习代码，涵盖从基础到高级的各种数据结构和算法。

## 📚 学习内容

### 第一部分：线性结构
- **LinearStructurePractice.java** - 线性结构练习
  - 动态数组（增删改查、扩容）
  - 单链表、双链表（增删改查、反转）
  - 栈（数组实现、链表实现）
  - 队列（普通队列、循环队列、双端队列）

### 第二部分：树结构
- **TreeStructurePractice.java** - 树结构练习
  - 二叉树遍历（前序、中序、后序、层序）
  - 二叉搜索树（BST）的增删改查
  - 堆（最大堆、最小堆）
  - 二叉树相关问题（最大深度、平衡二叉树、最近公共祖先）

### 第三部分：排序算法
- **SortingAlgorithmPractice.java** - 排序算法练习
  - **基础排序**：冒泡排序、选择排序、插入排序
  - **高级排序**：快速排序、归并排序、堆排序、希尔排序
  - **特殊排序**：计数排序、桶排序、基数排序

### 第四部分：查找算法
- **SearchAlgorithmPractice.java** - 查找算法练习
  - 顺序查找
  - 二分查找（递归、迭代）
  - 插值查找、斐波那契查找
  - 哈希查找（哈希表实现）

### 第五部分：动态规划
- **DynamicProgrammingPractice.java** - 动态规划练习
  - 斐波那契数列（递归、DP、空间优化）
  - 爬楼梯问题
  - 最长公共子序列（LCS）
  - 最长递增子序列（LIS）
  - 背包问题（0-1背包、完全背包）
  - 编辑距离
  - 最大子数组和
  - 硬币找零

### 第六部分：图算法
- **GraphAlgorithmPractice.java** - 图算法练习
  - 图的表示（邻接表、邻接矩阵）
  - 图的遍历（DFS、BFS）
  - 最短路径（Dijkstra、Floyd）
  - 最小生成树（Prim、Kruskal）
  - 拓扑排序
  - 环检测

## 💡 重点知识点

### 时间复杂度
- O(1)：常数时间
- O(logn)：对数时间（二分查找）
- O(n)：线性时间（遍历）
- O(nlogn)：线性对数时间（快速排序、归并排序）
- O(n²)：平方时间（冒泡排序）
- O(2^n)：指数时间（递归斐波那契）

### 空间复杂度
- O(1)：常数空间
- O(n)：线性空间
- O(n²)：平方空间

### 常用数据结构对比

| 数据结构 | 查找 | 插入 | 删除 | 适用场景 |
|---------|------|------|------|---------|
| 数组 | O(1) | O(n) | O(n) | 随机访问 |
| 链表 | O(n) | O(1) | O(1) | 频繁插入删除 |
| 栈 | O(n) | O(1) | O(1) | 后进先出 |
| 队列 | O(n) | O(1) | O(1) | 先进先出 |
| 二叉搜索树 | O(logn) | O(logn) | O(logn) | 有序数据 |
| 哈希表 | O(1) | O(1) | O(1) | 快速查找 |
