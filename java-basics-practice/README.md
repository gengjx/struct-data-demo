# Java基础阶段学习练习

本模块包含了Java基础阶段（第一部分）的完整练习代码，涵盖从语法基础到Lambda和Stream的所有知识点。

## 📚 学习路线

### 第一部分：Java语法基础
- **BasicSyntaxPractice.java** - 基础语法练习
  - 变量和数据类型
  - 运算符（算术、关系、逻辑、赋值）
  - 流程控制（if/else, switch, for, while, do-while）
  - 嵌套循环、break和continue

### 第二部分：数组和字符串
- **ArrayAndStringPractice.java** - 数组和字符串练习
  - 数组的声明、初始化、遍历
  - 数组的查找、排序、复制
  - 字符串的创建和常用方法
  - StringBuilder和StringBuffer

### 第三部分：面向对象编程
- **OOPPractice.java** - 面向对象练习
  - 类的定义和对象的创建
  - 封装（private、getter、setter）
  - 继承（extends、super、方法重写）
  - 多态（方法重载、方法重写、向上转型）
  - 抽象类和抽象方法

### 第四部分：接口、抽象类、内部类
- **InterfaceAndInnerClassPractice.java** - 接口和内部类练习
  - 接口的定义和实现
  - 接口的默认方法和静态方法
  - 抽象类的使用
  - 成员内部类、静态内部类、局部内部类、匿名内部类

### 第五部分：枚举和注解
- **EnumAndAnnotationPractice.java** - 枚举和注解练习
  - 枚举的定义和使用
  - 枚举的方法和属性
  - 自定义注解
  - 元注解的使用

### 第六部分：集合框架
- **CollectionFrameworkPractice.java** - 集合框架练习
  - List接口（ArrayList、LinkedList）
  - Set接口（HashSet、TreeSet、LinkedHashSet）
  - Map接口（HashMap、TreeMap、LinkedHashMap）
  - Queue接口（PriorityQueue）
  - 泛型的使用
  - Collections工具类

### 第七部分：I/O流
- **IOPractice.java** - I/O流练习
  - 字节流（InputStream、OutputStream）
  - 字符流（Reader、Writer）
  - 缓冲流的使用
  - 文件操作（File类）
  - 序列化和反序列化
  - NIO基础

### 第八部分：多线程
- **ThreadPractice.java** - 多线程练习
  - 线程的创建方式（Thread、Runnable）
  - 线程同步（synchronized、Lock）
  - 线程通信（wait、notify）
  - 线程池（ExecutorService）
  - 并发工具类（CountDownLatch、CyclicBarrier、Semaphore）

### 第九部分：异常处理和反射
- **ExceptionAndReflectionPractice.java** - 异常和反射练习
  - 异常的分类和处理
  - try-catch-finally、try-with-resources
  - throw和throws
  - 自定义异常
  - 反射的基本使用（Class、Method、Field）
  - 通过反射创建对象、调用方法

### 第十部分：Lambda和Stream
- **LambdaAndStreamPractice.java** - Lambda和Stream练习
  - Lambda表达式语法
  - 函数式接口（Predicate、Function、Consumer、Supplier）
  - 方法引用
  - Stream的创建和操作
  - Stream的中间操作（filter、map、sorted、distinct）
  - Stream的终止操作（collect、forEach、reduce）

## 🚀 使用方法

1. **打开练习类**
   - 每个知识点都有对应的练习类
   - 类名格式：`XxxPractice.java`

2. **完成练习**
   - 每个练习方法中都有`TODO`注释
   - 根据注释和要求完成代码
   - 可以参考类中的示例代码

3. **运行测试**
   - 运行每个类的`main`方法
   - 查看输出结果，验证是否正确

4. **深入学习**
   - 理解每个概念的原理
   - 尝试修改代码，观察不同结果
   - 完成综合练习题目

## 📝 学习建议

1. **循序渐进**
   - 按照顺序学习，前面的知识点是后面的基础
   - 不要跳步，确保每个知识点都理解透彻

2. **动手实践**
   - 不要只看代码，一定要自己写
   - 完成所有TODO任务
   - 尝试修改和扩展代码

3. **理解原理**
   - 不仅要会写，更要理解为什么这样写
   - 理解Java的设计思想
   - 思考不同的实现方式

4. **多做总结**
   - 每完成一个练习，总结要点
   - 记录易错点和注意事项
   - 画出知识点的思维导图

5. **综合应用**
   - 将多个知识点结合起来
   - 完成综合练习题目
   - 尝试实现小型项目

## 💡 练习技巧

- **遇到问题先思考**：不要立即查答案，先尝试自己解决
- **多写注释**：写代码时添加注释，帮助理解
- **运行调试**：多运行代码，观察结果
- **对比学习**：比较不同实现方式的优缺点
- **查阅文档**：遇到不清楚的API，查阅Java官方文档
